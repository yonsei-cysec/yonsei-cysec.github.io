Automated Vulnerability Discovery

The very first step towards having secure software systems is to find and fix (or sanitize) the vulnerabilities in them. This research project aims to develop new techniques to automatically scan software systems to find latent vulnerabilities. This project focuses on developing techniques to discover vulnerabilities more effectively, more efficiently, or both.

Exploit Mitigation

It is practically impossible to have perfectly secure software systems, even with all the pre-release testing (or sanitizing) efforts. Therefore, we need a defense mechanism that can protect software systems at run time so as to minimize the consequence of potential exploitation of vulnerabilities. This project aims to develop new, effective mechanisms that can mitigate the consequences exploits at run time.

Machine-Learning-based Reverse Engineering

Reverse engineering is a human-labor-intensive task. Many automated solutions have been proposed, and many rely on heuristics. Heuristics lend themselves as a useful approximation of decision procedures of expert reverse engineers; still, they are less than ideal. This project aims to develop ways to use state-of-the-art machine learning techniques to solve various reverse engineering problems.
